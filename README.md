# Flappy Bird Clone (Unity WebGL)

A simple Flappy Bird-style game built using Unity and C#.
This project demonstrates core game development concepts like physics, collision detection, spawning mechanics, and UI handling.

---

## Play the Game

Play here: https://isha2923.github.io/Flappy-Bird-Webgl/

---

## Tech Stack

* Unity (WebGL Build)
* C#
* GitHub Pages (Deployment)

---

## Features

* Smooth bird movement with gravity and jump physics
* Score tracking system
* Dynamic pipe spawning
* Collision detection (Game Over system)
* Parallax background effect
* Restart gameplay functionality

---

## Project Structure

FlappyBird/
├── Assets/

## Learnings

* Unity Game Loop (Update, Start, Awake)
* Physics using Rigidbody2D
* Object Instantiation and Destroy
* UI handling in Unity
* WebGL deployment

---

## How to Run Locally

1. Open Unity Hub
2. Click Open Project
3. Select the project folder
4. Open `SampleScene`
5. Click Play

---

## Deployment

The game is deployed using Unity WebGL and GitHub Pages.

Steps followed:

* Built project using WebGL
* Uploaded Build/, TemplateData/, and index.html
* Enabled GitHub Pages
